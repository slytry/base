# CHANGELOG


 CHANGELOG может быть сгенерирован скриптом при выполнении release. 

Все коммиты с последнего релиза:

```BASH
git log <last tag> HEAD --pretty=format:%s
```

Новые фичи в этом релизе:

```BASH
git log <last release> HEAD --grep feature
```

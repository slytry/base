### Сокращеная запись свойства объекта в JavaScript с ES6
Теперь (с ES6) если свойство и ключ совпадает, писать можно так
```js
let cat = 'Miaow';
let dog = 'Woof';
let bird = 'Peet peet';

let someObject = {
  cat,
  dog,
  bird
}

console.log(someObject);

//{
//  cat: "Miaow",
//  dog: "Woof",
//  bird: "Peet peet"
//}
```
 Раньше было только так (ES5)
 ```js
 var cat = 'Miaow';
var dog = 'Woof';
var bird = 'Peet peet';

var someObject = {
  cat: cat,
  dog: dog,
  bird: bird
}
 ```

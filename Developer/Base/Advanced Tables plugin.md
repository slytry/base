# Горячии клавиши



| Hotkey                                            | Action                      |
| ------------------------------------------------- | --------------------------- |
| <kbd>Tab</kbd>                                    | Next Cell                   |
| <kbd>Shift</kbd> + <kbd>Tab</kbd>                 | Previous Cell               |
| <kbd>Enter</kbd>                                  | Next Row                    |
| <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>D</kbd> | Open table controls sidebar |

# Горячие клавиши навигации

## OPENING SIDEBARS

| Hotkey                                            | Action                      |
| ------------------------------------------------- | --------------------------- |
| <kbd>Tab</kbd>                                    | Toggle Left Sidebar         |
| <kbd>Shift</kbd> + <kbd>Tab</kbd>                 | Toggle Right Sidebar        |
| <kbd>Enter</kbd>                                  | Next Row                    |
| <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>D</kbd> | Open table controls sidebar |
